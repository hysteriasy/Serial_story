页眉页脚组件示例页面

🎉 组件功能演示

页眉组件特性

  • 响应式导航栏，支持桌面端和移动端
  • 下拉菜单功能（作品展示菜单）
  • 用户登录状态显示和管理
  • 平滑滚动效果
  • 滚动时导航栏背景变化

页脚组件特性

  • 简洁的页脚布局
  • 版权信息和快速链接
  • 返回顶部按钮（滚动超过300px显示）
  • 响应式设计适配

🔧 使用方法

1. 引入组件文件

<script src="js/header.js"></script>
<script src="js/footer.js"></script>

2. 初始化组件

document.addEventListener('DOMContentLoaded', function() {
    // 初始化页眉
    if (typeof initHeader === 'function') {
        window.headerComponent = initHeader();
    }
    
    // 初始化页脚
    if (typeof initFooter === 'function') {
        window.footerComponent = initFooter();
    }
});

📋 测试功能

导航测试

点击导航栏中的"首页"、"关于我"等链接测试平滑滚动效果。

下拉菜单

悬停在"作品展示"菜单上查看下拉菜单效果。

移动端菜单

在移动设备或缩小浏览器窗口测试汉堡菜单。

返回顶部

滚动到页面底部，然后使用右下角的返回顶部按钮。

💡 提示

这个示例页面展示了如何在任何页面中集成页眉页脚组件。 组件会自动处理样式注入、事件绑定和响应式适配。 查看页面源代码了解具体的实现方法。

这里是用于测试滚动效果的内容区域。当您滚动页面时，可以观察到以下效果：
