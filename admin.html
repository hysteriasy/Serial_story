为什么需要 GitHub Token？

在 GitHub Pages 环境下，系统需要 GitHub Personal Access Token 来：

  • 保存和同步用户数据到 GitHub 仓库
  • 管理文件权限设置
  • 删除和修改仓库中的文件

如何创建 Token？

  1. 访问 GitHub Token 设置页面
  2. 点击 "Generate new token" → "Generate new token (classic)"
  3. 设置 Token 名称，如 "Serial_story_admin"
  4. 选择过期时间（建议选择较长时间或无过期）
  5. 重要：勾选以下权限：
    • repo - 完整的仓库访问权限
    • contents:write - 读写仓库内容
    • metadata:read - 读取仓库元数据
  6. 点击 "Generate token" 生成 Token
  7. 注意：Token 只显示一次，请立即复制保存
Token 格式：ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
